Java Collections Framework

Essential interfaces and implementations for data storage and manipulation

Interface
List<E>

An ordered collection that allows duplicate elements. Provides positional access.

List<String> list = new ArrayList<>();
list.add("Java");
list.get(0); // "Java"
Common Implementations:
  • ArrayList - Resizable array
  • LinkedList - Doubly-linked list
  • Vector - Thread-safe array
Time Complexity:

Access: O(1) Search: O(n) Insert: O(n)

Interface
Set<E>

A collection that contains no duplicate elements. Models mathematical set abstraction.

Set<Integer> set = new HashSet<>();
set.add(1);
set.add(1); // Duplicate ignored
Common Implementations:
  • HashSet - Hash table implementation
  • TreeSet - Red-black tree implementation
  • LinkedHashSet - Hash table + linked list
Time Complexity:

Add: O(1) Contains: O(1) TreeSet: O(log n)

Interface
Map<K,V>

An object that maps keys to values. Cannot contain duplicate keys.

Map<String, Integer> map = new HashMap<>();
map.put("count", 42);
int val = map.get("count");
Common Implementations:
  • HashMap - Hash table implementation
  • TreeMap - Red-black tree implementation
  • LinkedHashMap - Hash table + linked list
Time Complexity:

Get/Put: O(1) TreeMap: O(log n)

Interface
Queue<E>

A collection designed for holding elements prior to processing (FIFO).

Queue<String> queue = new LinkedList<>();
queue.offer("first");
queue.poll(); // "first"
Common Implementations:
  • LinkedList - Basic queue operations
  • PriorityQueue - Priority-based ordering
  • ArrayDeque - Resizable array implementation
Time Complexity:

Insert/Remove: O(1) PriorityQueue: O(log n)

Collections Utility

Static methods that operate on or return collections (sorting, searching, etc.)

List<Integer> numbers = Arrays.asList(3,1,2);
Collections.sort(numbers);
Collections.reverse(numbers);
Key Methods:
  • sort() - Sorts a list
  • binarySearch() - Efficient search
  • synchronizedXxx() - Thread-safe wrappers
  • unmodifiableXxx() - Immutable wrappers

Stream API (Java 8+)

Functional-style operations on collections (filter, map, reduce, etc.)

List<String> filtered = list.stream()
  .filter(s -> s.length() > 3)
  .map(String::toUpperCase)
  .collect(Collectors.toList());
Key Operations:
  • filter() - Predicate-based filtering
  • map() - Element transformation
  • reduce() - Aggregation operations
  • collect() - Terminal operation
